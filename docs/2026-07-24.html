AI Research & Models

AMD to invest up to $5 billion in Anthropic in a 2-gigawatt chip deal

AMD and Anthropic announced a strategic partnership in which Anthropic will deploy up to 2 gigawatts of AMD Instinct MI450 GPU systems starting in the first half of 2027, with AMD's investment tied to Anthropic hitting deployment milestones. The companies are also jointly optimizing AMD's ROCm software for Claude.

Why it matters — AMD's largest-ever bet on a single model developer, and a sign Anthropic is diversifying compute supply beyond Nvidia.

AI for Software Development

Anthropic launches Claude Security, a vulnerability scanner built into Claude Code

Anthropic released Claude Security in public beta for all Claude Code users, letting developers run terminal-based scans that trace data flows across files to catch multi-component vulnerabilities before code is committed, alongside patch suggestions.

Why it matters — Puts AI-powered vulnerability detection directly in the hands of the same developers using AI to write the code in the first place.

VS Code 1.130 adds a dedicated agent host and faster multi-file review

The July release lets AI agent sessions run in a separate process reachable from multiple VS Code windows, adds compact diffs and worktree support to the Agents window, and lets the model pre-screen tool-call risk to cut down approval interruptions.

GitHub's MCP server moves to the stateless MCP spec ahead of a protocol-wide cutover

GitHub updated its MCP server to support the next Model Context Protocol specification, which drops sessions and the initialize handshake; the protocol itself goes fully stateless on July 28.

Why it matters — Simplifies scaling MCP-connected coding agents, which most major IDEs and CLIs now depend on for tool access.

AMD launches 6th Gen EPYC "Venice" server CPUs, the industry's first HPC chips on 2nm

AMD formally introduced its EPYC 9006 "Venice" processor family at its Advancing AI 2026 event, built on TSMC's 2nm process with up to 256 cores per chip, a new SP7 socket, and claimed 70%-plus performance gains over the prior generation.

Why it matters — Server CPU capacity underpins every cloud and AI data center developers deploy to.

DeepSeek retires its legacy API model names today

DeepSeek's deepseek-chat and deepseek-reasoner API model aliases, which had been transparently routed to V4 Flash during a grace period, stop working entirely as of July 24, 2026, 15:59 UTC; callers must switch to explicit deepseek-v4-flash or deepseek-v4-pro model IDs.

Why it matters — Any integration still on the old model names breaks today with no fallback.

BitMEX to shut down after 11 years, ending Sept. 23

BitMEX, the exchange that pioneered the 100x-leverage perpetual swap in 2014, told users it will wind down operations by September 23, forcibly closing remaining open positions starting August 26; the company says a proof-of-reserves check shows customer assets are fully covered.

Why it matters — One of the products that defined crypto derivatives trading is exiting the market it created.

Bridge hacks drain $35 million from Bitcoin- and Ethereum-linked protocols in six hours

Three separate exploits hit AFX Trade's Arbitrum bridge (~$24.15M), the Verus-Ethereum bridge (~$7.54M, the same bug class as a May hack), and Bitcoin-scaling network B² Network's staking contract (~$3.86M) within a single six-hour window.

Why it matters — A reminder that bridges connecting bitcoin to other chains remain a recurring, unresolved attack surface.
