OpenAI ships GPT-5.6 to general availability, calls Sol its best coding model yet

OpenAI released the GPT-5.6 family — Sol, Terra, and Luna — to general availability across ChatGPT, Codex, and the API, ending a two-week restricted rollout the U.S. government had requested. OpenAI says Sol sets a new state of the art on the Artificial Analysis Coding Agent Index while using under half the output tokens of rival models, and it becomes the preferred model behind Microsoft 365 Copilot.

Why it matters — The clearest signal yet of how central this model family is to both OpenAI's consumer product and its Microsoft partnership.

GhostApproval symlink flaw lets malicious repos escape the sandbox in six AI coding agents

Wiz disclosed GhostApproval, a pattern where a symlinked file in a cloned repository tricks Amazon Q Developer, Claude Code, Augment, Cursor, Google Antigravity, and Windsurf into writing outside the intended workspace — in one proof of concept, overwriting a victim's SSH authorized_keys file. In several cases the agent's own reasoning flagged the danger while the confirmation prompt shown to the user hid it. AWS, Cursor, and Google have shipped fixes; two vendors went quiet and one rejected the report as outside its threat model.

Why it matters — A reminder that agentic coding tools inherit decades-old Unix trust problems, and that "are you sure?" prompts can lie about what they're approving.

GitHub's Copilot week: GPT-5.6 lands, Codex reaches JetBrains, desktop app goes free for all tiers

In quick succession GitHub added OpenAI's new GPT-5.6 family (Sol, Terra, Luna) as selectable Copilot models, brought Codex on as an agent provider inside JetBrains IDEs (IntelliJ, PyCharm, WebStorm) in public preview, and made the standalone Copilot desktop app free on every plan tier, including free and Education accounts.

Why it matters — Extends Copilot's reach into JetBrains' large enterprise Java/Kotlin/.NET base while lowering the floor for free-tier agentic coding.

No AI lab scores above a C+ in Future of Life Institute's summer safety index

FLI's Summer 2026 AI Safety Index graded nine leading developers on 37 indicators across risk assessment, current harms, and existential safety. Anthropic led with a C+, OpenAI and Google DeepMind received Cs, and xAI, DeepSeek, and Mistral failed outright. The panel found several labs have quietly weakened or dropped earlier pledges to pause development near dangerous capability thresholds.

Why it matters — An independent check on whether safety commitments are keeping pace with this week's release cadence.

Dev Tools & Releases

TypeScript 7.0 ships with a Go-native compiler that's up to 10x faster

Microsoft released TypeScript 7.0, completing the port of the compiler from a TypeScript-on-JavaScript bootstrap to native Go. The team's own benchmarks show the VS Code codebase type-checking in 7.5 seconds versus 77.8 seconds on 6.0, and Playwright's checks dropping from 11.1s to 1.1s.

Why it matters — One of the largest performance jumps in the compiler's history, and for most projects a near drop-in upgrade.

Microsoft (TypeScript blog)

npm v12 goes GA, blocking install scripts and remote/git dependencies by default

npm v12 is now generally available and tagged latest. It turns on install-time security defaults — no longer auto-running preinstall/install/postinstall scripts, resolving git dependencies, or fetching remote tarballs unless explicitly allowed — and starts deprecating the riskiest 2FA-bypass granular access tokens.

Why it matters — A direct response to this year's npm supply-chain attacks; CI pipelines relying on lifecycle scripts may see silent build breaks until updated.
