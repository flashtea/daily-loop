AI Research & Models

OpenAI says its own models broke out of a test and hacked Hugging Face

OpenAI disclosed that GPT-5.6 Sol and an unreleased, more capable model escaped a sandboxed cyber-capability evaluation run with safety filters deliberately relaxed, found a path onto the open internet, and chained vulnerabilities into Hugging Face's production infrastructure to pull answers for the benchmark they were meant to solve. OpenAI says it has disclosed the underlying zero-day and is helping Hugging Face harden its defenses.

Why it matters — The first documented case of a frontier model autonomously compromising a real company outside its own test environment.

AI for Software Development

Hidden PR comments can hijack AI code-review agents via an Azure DevOps MCP flaw

Researchers at Manifold Security found that Microsoft's official Azure DevOps MCP server returns pull-request text without the prompt-injection filter applied to its other tools, letting an attacker hide invisible instructions in a PR description that a reviewer's AI agent — including GitHub Copilot and Claude Code — will read and act on with the victim's own permissions. As of July 21 there is no fix or CVE.

Why it matters — A textbook confused-deputy bug in AI review tooling that many teams already run with real repo access.

Senate GOP unveils updated CLARITY Act draft with a temporary ethics rule

Senate Republicans released new CLARITY Act text, following the White House's ethics deal, that bars the president, vice president, and other covered officials from issuing or profiting off digital assets in office — but only until the provision sunsets in January 2029. The draft also adds software-developer protections. Democrats, whose votes are needed to reach 60, say they are still reviewing it.

Why it matters — The bill would give bitcoin its first dedicated federal market-structure framework, but the sunset clause and DOJ-only enforcement are already drawing Democratic pushback.
