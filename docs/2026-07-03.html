AI Research & Models

UN scientific panel warns AI safeguards can't keep pace with capabilities

The Independent International Scientific Panel on AI, co-chaired by Turing Award winner Yoshua Bengio and Nobel Peace laureate Maria Ressa, released its first preliminary report, concluding that science currently cannot guarantee advancing AI capabilities won't cause catastrophic harm. The findings feed into the inaugural UN Global Dialogue on AI Governance in Geneva on July 6–7.

Why it matters — The first independent, government-agnostic scientific assessment of frontier AI risk, timed to directly shape an intergovernmental governance summit.

AI for Software Development

Cognition launches Devin Security Swarm, an autonomous vulnerability-hunting agent

Cognition's new product deploys swarms of Devin agents to triage and fix security vulnerabilities at scale. In testing, it found 36 of 50 real-world GHSA vulnerabilities at roughly 30% lower cost per finding than the next most accurate alternative.

Why it matters — Pushes AI coding agents beyond writing code and into the unglamorous, backlog-heavy work of finding and patching vulnerabilities.

Cascade hits hard end-of-life as Devin Desktop's Agent Command Center becomes the default

Windsurf's Python-based Cascade agent reached its scheduled end-of-life on July 1 with no grace period, fully replaced by the Rust-based Devin Local inside Devin Desktop's new Agent Command Center, a Kanban-style board for managing local and cloud coding agents that supports the open Agent Client Protocol.

Why it matters — Any CI pipeline or script that called Cascade by name stopped working overnight; teams needed to repoint automation before the cutover.

Kimi K2.7 Code becomes the first open-weight model in GitHub Copilot's picker

Moonshot AI's Kimi K2.7 Code, a 1-trillion-parameter open-weight coding model, is now generally available as a selectable model in GitHub Copilot, rolling out to Pro, Pro+ and Max plans.

Why it matters — The first time an open-weight coding model has sat alongside closed frontier models in Copilot's default picker.

OpenAI's new biology benchmark exposes a wide AI research-judgment gap

OpenAI introduced GeneBench-Pro, a benchmark testing whether AI models can exercise research-level judgment on computational biology problems, such as distinguishing genuine signal from noise. Even top frontier models scored only around 32%.

Why it matters — A concrete, primary-source data point on how far current models are from genuine scientific judgment, not just fact recall.

Dev Tools & Releases

GitHub Issue fields reach general availability

GitHub's structured, typed metadata for issues (priority, effort, dates and custom fields) is now generally available for Free, Team, Enterprise and data-residency plans, and will ship in GitHub Enterprise Server 3.23. Field values now also show directly in the repository issue list.

Why it matters — Over 40,000 organizations adopted the feature during preview; GA makes searchable, reportable issue metadata a default part of GitHub project tracking.

Dependency-confusion campaign publishes 25 malicious npm packages mimicking internal tooling

A newly created npm account, marketfront, batch-published 25 packages under the @marketfront scope in a roughly three-minute window on July 1, each carrying a postinstall credential harvester and a reused 'Internal package' README lure. Researchers linked the campaign, via naming patterns and payload structure, to a series of disposable accounts running the same playbook.

Why it matters — Dependency-confusion attacks that mimic a specific company's private package names can silently compromise any build missing the real internal package.

Bitcoin

Bitcoin rallies past $61,000 as Fed's Warsh eases inflation talk and jobs data disappoints

Bitcoin climbed more than 4% to briefly reclaim $62,000 after Fed Chair Kevin Warsh told the ECB forum in Sintra that inflation risks have come down, and June's US jobs report showed only 57,000 jobs added versus the roughly 115,000 forecast. Wage growth accelerated to 3.5% year-on-year, though, leaving the rate-cut outlook mixed.

Why it matters — The first notably dovish signal from Warsh since a hawkish June dot plot triggered weeks of ETF outflows, reviving the 'debasement trade' narrative.

Metaplanet buys 2,823 more Bitcoin, becomes world's third-largest corporate holder

The Tokyo-listed firm added 2,823 BTC for about $170.7 million, bringing its total to 43,000 BTC (worth roughly $2.6 billion) and moving it past other large holders into third place globally, behind Strategy and Twenty One Capital.

Why it matters — Corporate treasury accumulation continues even as US spot ETFs post record outflows, underscoring a split between institutional channels.

CLARITY Act's July 4 deadline quietly slips as Senate heads into recess

The Senate adjourned until July 13 without resolving disputes over ethics rules tied to Trump's crypto holdings, DeFi oversight and anti-money-laundering provisions, pushing any floor vote on the crypto market-structure bill past its symbolic July 4 target toward the tighter pre-August-recess window.

Why it matters — The bill sets the CFTC/SEC jurisdictional split for digital assets; another missed deadline raises the odds it slips to 2027.
