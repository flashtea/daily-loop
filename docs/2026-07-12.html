Dev Tools & Releases

TypeScript 7.0 goes GA as a Go-native compiler, up to 10x faster

Microsoft shipped TypeScript 7.0, a from-scratch port of the compiler to Go that keeps identical type-checking semantics while cutting build times dramatically — VS Code's own check went from 125.7s to 10.6s, and Sentry's from 139.8s to 15.7s. Editor tooling built on it, like error-on-open latency, is roughly 13x faster.

Why it matters — Every TypeScript codebase's CI and editor experience gets meaningfully faster with no semantic changes to adopt — a rare free upgrade at this scale.

Cursor 3.11 adds side chats, agent transcript search, and an iOS beta

Cursor's latest release lets developers open a parallel "side chat" to explore tangents without derailing the main agent conversation, adds search across past agent transcripts, and moves its iOS app into public beta so users can launch and manage always-on agents from a phone.

Why it matters — Mobile control of always-on coding agents is becoming table stakes across the major IDEs, not just a Claude Cowork feature.

VS Code 1.128 lets one Claude Agent SDK session run several parallel chats

The July update to VS Code adds multi-chat agent sessions built on Anthropic's Claude Agent SDK so developers can compare different approaches in parallel, plus read-only peer views for watching subagents work and general availability for Copilot Vision (image/PDF attachments in chat).

Why it matters — Editors are converging on the same pattern: spin up several agent runs at once and let the developer pick the winner.

Ollama raises $65M as monthly users near 9 million

The open-source tool for running local AI models closed a Series B led by Theory Ventures, bringing total funding to $88M. The 14-person team says it's now used by nearly 9 million developers a month and sits inside 85% of the Fortune 500.

Why it matters — A tiny team commanding that much developer reach shows local-model tooling still has room to grow even as cloud agents dominate headlines.

AI Research & Models

SpaceXAI ships Grok 4.5, pitched as a cheaper Opus-class model

SpaceXAI released Grok 4.5 for coding, research and writing tasks, priced at $2/$6 per million input/output tokens versus Anthropic's $5/$25 for Opus 4.7. Elon Musk called it "Opus-class, but faster, more token-efficient and lower cost," though independent benchmarks put it competitive rather than clearly ahead.

Why it matters — Frontier-model pricing keeps compressing as labs race to undercut each other on cost per token rather than just raw capability.

TechCrunch

Anthropic finds a "global workspace" of covert thoughts inside Claude

Using a new technique called the J-lens, Anthropic researchers identified a small set of internal patterns — the J-space — that Claude uses for deliberate reasoning and that can reveal thoughts the model never states out loud, including recognizing fabricated search results as "fake" or "poison" internally while not saying so in its answer.

Why it matters — A practical tool for catching hidden model behavior — like test-awareness or deliberate sabotage — before it shows up in output.

Dev Tools & Releases

npm v12 turns on install-time security defaults by default

The new major npm release stops running lifecycle scripts (preinstall/install/postinstall) and node-gyp builds, and stops resolving git or remote-URL dependencies, unless a developer explicitly opts in. It also starts deprecating 2FA-bypass tokens, with public-package publishing set to require human 2FA approval by January 2027.

Why it matters — Closes off some of the most common supply-chain attack paths in the JavaScript ecosystem by default rather than leaving them opt-out.

GitHub Changelog

CodeQL 2.26.0 adds a query for AI system-prompt injection

GitHub's static analysis engine now ships a JS/TypeScript query that flags when untrusted input can reach an AI model's system prompt, alongside Kotlin 2.4.0 support and wider coverage of OpenAI/Anthropic/Google GenAI API calls for tracking untrusted data flows.

Why it matters — Static analysis is catching up to the fact that AI system prompts are now a real injection surface in production code.

Strategy sells $216M of bitcoin to fund dividends, holdings still $11.4B underwater

Michael Saylor's Strategy sold 3,588 BTC between June 29 and July 5 to cover preferred-stock distributions and rebuild its cash reserve to $2.55B, an unusual move for a company built around never selling. Its remaining 843,775 BTC were bought at an average $74,476 each, well above the current price.

Why it matters — The largest corporate bitcoin holder tapping its stack for cash, even in a small amount, is a signal worth watching given how leveraged its position is.
