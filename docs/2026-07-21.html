Alibaba previews Qwen3.8-Max, a 2.4-trillion-parameter model it claims trails only Claude Fable 5

Alibaba unveiled a preview of Qwen3.8-Max, a 2.4T-parameter multimodal model handling text, images, video, and documents with a 1M-token context window, days after Moonshot AI's Kimi K3 launch. Alibaba shares rose as much as 5.4% on the announcement.

AI for Software Development

GitHub Code Quality reaches general availability, bundling AI code review into a paid product

GitHub Code Quality moved from preview to general availability on GitHub Enterprise Cloud and Team, pairing CodeQL's deterministic analysis with AI-assisted detection and Copilot Autofix, plus org-wide dashboards and coverage-based quality gates.

Why it matters — It's priced at $10 per active committer/month plus usage-based AI charges — the clearest sign yet that GitHub is monetizing AI-assisted review at scale, no Copilot subscription required.

GitHub Changelog

Copilot code review now reads instructions from your feature branch, adds CLAUDE.md/GEMINI.md support

GitHub updated Copilot code review to pull custom instructions from a pull request's head branch instead of its base branch, and to automatically pick up REVIEW.md, GEMINI.md, and CLAUDE.md files alongside its own config format.

Why it matters — Lets teams iterate on review guidelines in a feature branch, and standardizes on the config files rival agent tools already use.

China approves Apple Intelligence for launch, built on Alibaba's Qwen models

The Cyberspace Administration of China approved Apple Intelligence for release in the country, with Alibaba's Qwen models integrated into iOS, iPadOS, macOS, and visionOS to power on-device text and image understanding and generation.

Why it matters — Ties Apple's AI rollout directly to a domestic Chinese model rather than a Western one, a regulatory condition unique to that market.

Future of Life Institute's Summer 2026 AI Safety Index: no lab scores above a C+

FLI's latest safety index graded nine AI labs across 37 indicators; Anthropic led at C+ (2.66), followed by OpenAI and Google DeepMind at C, while xAI, DeepSeek, and Mistral scored F. Reviewers flagged a broad retreat from earlier safety pledges and a pivot toward military AI applications.

Why it matters — Existential-safety scoring was the weakest domain industry-wide, with no company exceeding a C-.

Dev Tools & Releases

GitHub's 'pwn request' fix for actions/checkout now enforced on every supported major version

GitHub's backport blocking pull_request_target and workflow_run workflows from checking out unreviewed fork PR code — previously opt-in via actions/checkout v7 — became mandatory across v2 through v6 on July 20, closing a pattern behind multiple past supply-chain incidents.

Why it matters — Workflows pinned to a floating tag like @v4 pick up the protection automatically; those pinned to a SHA or exact version need to upgrade manually.
