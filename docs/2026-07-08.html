AI Research & Models

Every major AI lab still fails independent safety grading, new index finds

The Future of Life Institute's Summer 2026 AI Safety Index graded nine leading AI companies on 37 indicators; Anthropic topped the field with a C+, OpenAI and Google DeepMind got C's, and xAI, DeepSeek, and Mistral each received an F. No company scored above a C- on existential-safety planning.

Why it matters — Panelist Stuart Russell said labs have walked back earlier pledges to gate releases on safety measures and are 'now planning releases even if demonstrably unsafe' — a warning sign as these same labs race to ship ever more capable coding and agentic models.

AI for Software Development

GitHub Copilot's desktop app rolls out to every plan, including Free

GitHub made the standalone Copilot app generally available on macOS, Windows, and Linux for all Copilot plans, including Copilot Free and GitHub Education, having previously been limited to paid tiers.

Why it matters — Removes the last paywall between casual developers and Copilot's agent-native desktop surface.

GitHub Changelog

Kimi K2.7 Code becomes the first open-weight model in Copilot's picker

GitHub made Moonshot AI's Kimi K2.7 Code generally available as a selectable model in Copilot, and extended it to Copilot Business and Enterprise plans, where administrators must opt in before employees can use it.

Why it matters — The first non-proprietary model in Copilot's picker gives enterprises a lower-cost coding option alongside GPT and Claude models.

GitHub Changelog

Anthropic's Claude Cowork moves off the desktop, onto web and mobile

Claude Cowork, Anthropic's general knowledge-work agent, launched as a desktop-only app in January but is now available on web and mobile for Max subscribers, letting users start a task at a desk and check progress from a phone.

Why it matters — Anthropic's own usage data shows software development is now just 8.7% of Cowork's workload, versus 33.4% for business-process tasks — a sign coding-agent vendors are chasing the rest of the office.

Meta ships its first image and video generation models

Meta Superintelligence Labs launched Muse Image and Muse Video, its debut media-generation models, rolling out in the Meta AI app, on meta.ai, and in Instagram Stories in the US, with WhatsApp and Facebook support coming soon.

Why it matters — Pairs with Meta's Muse Spark reasoning model to plan and generate images in one step, Meta's clearest bid yet to compete with OpenAI and Google on creative AI.

Dev Tools & Releases

GitHub Models will be fully retired on July 30

GitHub confirmed its GitHub Models playground, model catalog, inference API, and bring-your-own-key support will shut down entirely on July 30, 2026, giving developers three weeks to migrate off the service.

Why it matters — Teams prototyping against GitHub Models' free inference API need a migration plan before the hard cutoff.

GitHub Changelog

Copilot CLI in GitHub Actions no longer needs a personal access token

GitHub Copilot CLI can now authenticate in GitHub Actions using the workflow's built-in GITHUB_TOKEN instead of a stored personal access token, provided the workflow grants copilot-requests: write permission.

Why it matters — Cuts a common source of long-lived-credential risk in CI pipelines that call Copilot CLI.

GitHub Changelog

GitHub issue fields reach general availability

Custom issue fields, letting teams attach structured metadata to issues, are now generally available across GitHub Free, Team, Enterprise, and GitHub Enterprise Cloud with data residency plans.

Why it matters — Brings GitHub Issues closer to dedicated project-tracking tools without leaving the platform.

Bitcoin shrugs off Strategy's $216M sale but futures positioning stays thin

Bitcoin touched $64,400 overnight before settling back near $63,170, ending roughly flat on the day but still up about 6% for the week; the market absorbed Strategy's $216 million bitcoin sale without disrupting the recovery, even as CME futures open interest sat at a 32-month low and downside options protection neared record highs.

Why it matters — Thin futures activity and elevated hedging demand suggest traders remain unconvinced the bounce off June's lows will hold.

CoinDesk

US Strategic Bitcoin Reserve stalls as Treasury and Commerce fight over control

Sixteen months after President Trump ordered the creation of a Strategic Bitcoin Reserve, the White House says it is still evaluating the 'best structure' for the reserve, as Treasury and Commerce both seek to oversee the roughly 300,000 BTC (about $21 billion) the government already holds. Congress has not passed the legislation officials say is needed to finalize it.

Why it matters — An interagency turf war is holding up clarity on what the US government plans to do with its multibillion-dollar bitcoin holdings.
